# Alla Taskaeva Website

Промо-сайт на **Next.js 14 + Tailwind CSS**.

## Локальный запуск

```bash
npm install
npm run dev
```

Открыть: `http://localhost:3000` (или порт из лога).

## Продакшн-сборка

```bash
npm run build
npm run start
```

## Деплой на Vercel (самый простой)

1. Открой репозиторий на GitHub: `https://github.com/DanilKindly/allatask`
2. В Vercel нажми **Add New Project**
3. Импортируй `DanilKindly/allatask`
4. Нажми **Deploy**

Vercel сам определит Next.js и выдаст публичную ссылку вида:
`https://allatask-<hash>.vercel.app`

## Структура

- `app/` — роуты и layout
- `components/` — секции страницы
- `public/images/` — изображения
